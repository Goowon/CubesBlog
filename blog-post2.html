Tools to Help You Design Your Island

Click on the names of each tool to go to the tools' website!

3D Island Planner

Anyone that took a stab at terraforming knows how time consuming it can be, especially when each block takes around a second and a half to edit. This tool is great because it allows you to quickly edit the map in its editor and supports placing buildings and adding rivers to the terrain.

Happy Island Designer

As great as the first tool is, it might be slow if you're not running it on a great pc. The second tool is a great pixel editor of the island that allows for editing the island according to the minimap!

I would recommend using the Happy Island Designer to design the overall idea of your island and then go deep into the details if possible using the 3D Island Planner.
